How to install PageSigner on Chrome


First, make sure your Chrome is up to date (it will probably work with recent versions, but this is advisable).

Download this zip file. This is PageSigner extension for Chrome

Open 2 new tabs 'chrome://extensions' and 'chrome://downloads' (without the quotes).

In chrome://downloads grab 103.zip and drag it to the Extensions tab. When Extensions tab becomes active, drop the zip file inside of it.



When you've done that, you'll see the pagesigner entry in the extensions tab created. You'll also see the pagesigner logo (blue signing logo) in the top right of the Chrome window. Head over to that and click it, and you'll be prompted thusly:



Go ahead and do what it says. This will bring up the 'Add helper app to Chrome' dialog (with some fancy screenshots) like this:



Click 'Add to Chrome' (the big blue button). You'll get prompted:



Click 'Add'. This will bring up a new dialog 'Launch'. You can ignore (close) that. You're done! PageSigner is installed and should be working immediately. To test, browse to a new page (don't use a page that was already loaded before PageSigner), and you should be able to choose 'Notarize this page' from the PageSigner menu:



Happy notarizing!
