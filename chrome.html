How to install PageSigner on Chrome


First, make sure your Chrome is up to date (it will probably work with recent versions, but this is advisable).

Download PageSigner extension for Chrome

You're done! PageSigner is installed and should be working immediately. To test, browse to a new page (don't use a page that was already loaded before PageSigner), and you should be able to choose 'Notarize this page' from the PageSigner menu:



Happy notarizing!
